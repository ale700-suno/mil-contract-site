# CONTRACT RF

**Официальный сервис для подачи анкет и контрактов**

---

## О проекте

CONTRACT RF — веб-сайт + Telegram-бот для удобной подачи анкет и оформления контрактов.

### Основные возможности:
- Подача анкет через удобную форму
- Интеграция с Telegram-ботом (@ContractRFBot)
- Отправка заявок в группу
- Современный тёмный дизайн

---

## Технологии

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Three.js** (3D элементы)
- **Telegram Bot API**
